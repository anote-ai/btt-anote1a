# Benchmark Testing Dataset

**Name:** Multilingual Benchmark Testing Dataset

**Version:** 1.0

**Created:** 2025-11-29

**Languages:** ES (Spanish), HE (Hebrew), JA (Japanese), KO (Korean)

**Models:** ChatGPT, Claude, Gemini

## Description
Multilingual QA-citation testing dataset with factual, reasoning, and ambiguous/challenging questions generated by multiple LLMs per language for benchmarking multilingual question-answering and citation fidelity.

## Structure
- 4 languages × 3 models × 6 batches per model = 72 batch files
- 6 items per batch file = 432 total QA-citation pairs
- Question types: factual, reasoning, ambiguous
- Difficulty levels: easy, medium, hard

## Use Cases
- Cross-lingual model evaluation
- Citation grounding checks
- Multilingual QA benchmark testing
- Model comparison across languages
