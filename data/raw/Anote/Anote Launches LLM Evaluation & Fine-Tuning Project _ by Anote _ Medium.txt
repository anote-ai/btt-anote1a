*Anote Launches LLM Evaluation & Fine-Tuning Project*

Anote <https://anote.ai/#top> (Headquarters: New York, USA; CEO: Natan
Vidra) proudly announces the launch of an initiative to evaluate and
improve large language models (LLMs).

This project centers on *Anote’s* <https://github.com/nv78/OpenAnote/
tree/main> recently released end-to-end MLOps platform designed to
democratize access to human centered AI development. From data
annotation to fine-tuning, model inference, evaluation, and integration,
Anote supports the full lifecycle of LLM implementation.

Press enter or click to view image in full size

/Title: Human Centered AI Methodology
Description: Anote is an end to end MLOps platform that enables you to
obtain the best large language model for your data. On Anote, we provide
an evaluation framework to compare zero shot LLMs like GPT, Claude,
Llama3 and Mistral, with fine tuned LLMs that are trained on your domain
specific training data (via supervised, unsupervised and RLHF fine
tuning). We provide a data annotation interface to convert raw
unstructured data into an LLM ready format, and incorporate subject
matter expertise into your training process to improve model accuracies.
End users can route / integrate the best LLM into their own, on premise,
private chatbot, as well as interact with our software development kit
for fine tuning./

*Addressing Critical LLM Challenges*

Multilingual LLMs are often dominated by English data (60–70%), while
other languages typically accounts for only 3–5% of training corpora.
Anote seeks to close this gap by empowering developers and researchers
with the tools and infrastructure needed to build and benchmark superior
language AI.

This project addresses several major problems:

 1. *Insufficient and low-quality training data* — Anote helps generate
    high-quality, large-scale training datasets.
 2. *Lack of objective LLM benchmarks* — This initiative introduces the
    first publicly shared LLM evaluation datasets / metrics and model
    leaderboards.
 3. *Limited access to fine-tuning infrastructure* — Participants can
    utilize Anote’s platform to build and deploy custom LLMs using their
    own data.

*What Makes Anote Unique*

  * *End-to-End MLOps Workflow*: Annotation, fine-tuning (via LoRA,
    QLoRA, RLHF), model inference, evaluation, and integration in a
    single platform.
  * *Multi-Model Benchmarking*: Compare GPT-4o, Claude 3.5, Llama 3,
    Mistral, and fine-tuned models using real multi-lingual data.
  * *Advanced Evaluation Framework*: Assess models using Cosine
    Similarity, Rouge-L, LLM Eval, Answer Relevance, and Faithfulness.
  * *Task Versatility*: Supports text classification, named entity
    recognition (NER), full-document QA, and prompt-based QA.
  * *Integration Ready*: API and SDK access to deploy optimized models
    into production environments.

/Title: Data Labeler

state of the art few shot learning to make high quality predictions with
a few labeled samples. You can use this with 4 main steps:/

 1. */Upload/*/— Create a new text based dataset/
 2. */Customize/*/— Add the categories, entities or questions you care
    about/
 3. */Annotate/*/— As you annotate a few edge cases, the LLM actively
    learns to predict the rest/
 4. */Download/*/— Download the resulting CSV / export the fine tuned
    LLM as an API endpoint/

*Envisioning the Future of Multi-Lingual AI*

Through this initiative, participating partners will:

  * Build models optimized for language and context
  * Develop internal tools and products using fine-tuned LLMs
  * Contribute to the first public benchmark datasets for NLP
  * Showcase their results through Anote’s leaderboard and co-authored
    research

*Application Summary*

  * *Number of Participants*: Limited to 5 partners
  * *Project Timeline*: June 1 — October 1, 2025
  * *Application Deadline*: May 17, 2025
  * *How to Apply*: Please contact the email addresses below
